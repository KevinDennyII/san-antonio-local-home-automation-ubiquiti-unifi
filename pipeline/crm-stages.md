# CRM Stages — Lead to Concierge

**Pipeline:** Lead → Survey → Proposal → Install → Concierge  

Use HubSpot, Notion, Airtable, or a spreadsheet. Every realtor intro gets a card the same day.

---

## Stages

| Stage | Entry criteria | Exit criteria | Owner |
|-------|----------------|---------------|-------|
| **Lead** | Warm intro or inbound | Survey booked or disqualified | Provider |
| **Survey** | Survey on calendar / fee collected | Survey complete + notes filed | Provider |
| **Proposal** | SOW draft sent | Signed SOW + deposit **or** lost | Provider |
| **Install** | Deposit cleared; gear ordered | Handoff signed | Provider + low-voltage sub |
| **Concierge** | Handoff done | Active retainer **or** declined (churn reason logged) | Provider |

---

## Stage SLAs (internal)

| Transition | Target |
|------------|--------|
| Lead → Survey booked | 3 business days |
| Survey → Proposal sent | 2 business days |
| Proposal → decision | 14 days (nudge at day 7) |
| Deposit → install start | ≤ 21 days (gear lead times) |
| Install start → handoff | Per SOW (typically 2–5 on-site days) |
| Handoff → Concierge billing | Day 80 reminder for opt-in after included 90 |

---

## Lost reasons (pick one)

Wrong budget · Timing / move-in · Chose competitor · Wi‑Fi only DIY · Alarm vendor conflict · No response · Other

---

## Realtor visibility

Share **stage + next step** only (not full margins). Weekly 15-minute sync during first 90 days.
