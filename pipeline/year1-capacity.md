# Year-1 Capacity Plan

**Market:** San Antonio / Texas Hill Country  
**Constraint:** Quality for million-dollar homes > volume  

---

## Target throughput

| Period | Installs | Notes |
|--------|----------|-------|
| Days 1–30 | 0 full installs | Foundation: agreement, demo rack, insurance, supply |
| Days 31–60 | 2 pilots | Discounted; metrics mandatory |
| Days 61–90 | 1 full-price B/C | Concierge default on |
| Months 4–12 | **1–2 installs / month** | Cap unless second tech hired |
| Year-1 total (stretch) | ~10–14 installs | Plus growing Concierge MRR |

**Concierge MRR goal (illustrative):** 20 homes × $250 = **$5,000/mo** — prioritize retention over cramming installs.

---

## Calendar load per install (Package B typical)

| Role | Hours (indicative) |
|------|--------------------|
| Survey | 2–4 |
| Design / proposal | 2–3 |
| Low-voltage cabling (sub) | 8–24 (site-dependent) |
| Network commission | 4–8 |
| HA + devices | 6–12 |
| Handoff | 1–2 |
| Provider project mgmt | 3–5 |

**Rule:** Do not book overlapping day-1 rack bring-ups for two estates in the same week during year 1.

---

## Capacity kill switches

Pause new **Proposal** stage entries when:

- [ ] Two installs already in **Install** stage  
- [ ] Open P1 tickets > 0 for > 48 hours  
- [ ] Concierge client count > 15 without part-time help  
- [ ] Low-voltage sub unavailable for 3+ weeks  

---

## Hiring / subcontract triggers

| Trigger | Action |
|---------|--------|
| >2 installs/month demand sustained 2 months | Vet second low-voltage crew or junior tech |
| Concierge >20 homes | Part-time support for P3 tickets |
| Package C demand | Separate LLM-capable hardware budget + skills time |

---

## Q1 focus metrics

| Metric | Target |
|--------|--------|
| Pilot completion | 2 |
| Full-price closes | ≥1 by day 90 |
| Concierge attach rate | ≥80% of new installs after included period |
| Gross margin after referral | 35–45% |
| Avg tickets / home in first 14 days | Declining after pilots |

Review capacity monthly with realtor pipeline forecast (listings / closings).
